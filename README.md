# Earth Globe

Interactive 3D Earth with day/night cycle, 100+ major cities, and location search. Drag to rotate, scroll to zoom, click cities for info.

**[Live Demo](https://zebiv.com/earth-globe/)**

## Features

- Day/night texture blending with realistic terminator
- Sun position based on month and time of day
- 100+ major world cities with population data
- City search via OpenStreetMap Nominatim
- Custom city pins saved to localStorage
- Adjustable city count, labels, and time controls

## Tech

- WebGL (no libraries)
- Vanilla HTML/CSS/JavaScript
- Custom GLSL shaders for day/night blending
- Earth textures from Wikimedia Commons
